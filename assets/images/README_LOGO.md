# Logo Oluşturma

RDC logosu için:

1. SVG dosyası: `assets/images/logo.svg` (oluşturuldu)
2. PNG'ye çevirmek için:
   - Online: https://svgtopng.com/ veya https://convertio.co/svg-png/
   - Veya ImageMagick: `convert logo.svg logo.png`
   - Veya Inkscape: `inkscape logo.svg --export-filename=logo.png`

Logo özellikleri:
- Metin: "RDC" (büyük harfler)
- Renk: Mavi (#2196F3)
- Arka plan: Transparan
- Boyut: 200x200px
